# Naming Conventions

## Audio

Format:

`YYYY-MM-DD_hhmm_topic.m4a`

Example:

`2026-03-30_1930_first-job-story.m4a`

## Transcript

Format:

`YYYY-MM-DD_hhmm_topic.md`

The script writes transcripts in that format when the source audio filename matches the audio convention.

## Chapters

Format:

`chapter-01-childhood.md`

## Slugs

- use lowercase
- use hyphens instead of spaces
- avoid punctuation unless it is required for meaning
- keep titles specific enough to search later
