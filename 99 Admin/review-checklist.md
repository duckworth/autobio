# Review Checklist

Use this when processing a transcript out of [[00 Inbox]].

## Triage

- [ ] Decide whether this is `story`, `timeline`, or `guidance`
- [ ] Decide whether it is worth keeping
- [ ] Rename the note if the content deserves a better title
- [ ] Add tags that will help retrieval later

## Cleanup

- [ ] Correct major transcription errors only
- [ ] Preserve uncertainty instead of smoothing it over
- [ ] Mark unclear names, places, and dates
- [ ] Pull out any strong scenes or chapter-worthy passages

## Extraction

- [ ] Add timeline facts to [[03 Timeline/master-timeline|Master Timeline]]
- [ ] Add unresolved chronology to [[03 Timeline/open-questions|Open Questions]]
- [ ] Update or create people notes
- [ ] Update or create place notes
- [ ] Update or create theme notes

## Gap Handling

- [ ] Add unanswered questions to [[99 Admin/gap-log|Gap Log]]
- [ ] Turn each meaningful gap into a specific future memo prompt
- [ ] Mark whether the transcript should trigger an immediate follow-up recording

## Drafting Readiness

- [ ] Decide whether this note supports an existing chapter plan
- [ ] Decide whether it should start a new chapter candidate
