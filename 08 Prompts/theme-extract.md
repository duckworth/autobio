# Theme Extract Prompt

Goal:

- identify recurring themes
- connect memories across time
- preserve ambiguity where the source is ambiguous

Suggested instruction:

```text
Identify the memoir themes present in this transcript. Prefer themes that recur across multiple life periods such as work, family, money, movement, identity, loss, ambition, faith, or reinvention. Use short notes that can be linked into a theme note later.
```
